
# RÉFÉRENCE COMPLÈTE DES PARAMÈTRES LAVA (Version 2026)

Ce document liste l'ensemble des paramètres modifiables via la ligne de commande pour les scripts `lava_loop_primer.pl` (Mode Classique + Loop) et `lava_stem_primer.pl` (Mode Stem).

---

## 1. Entrée / Sortie (Obligatoires)

| Paramètre | Description |
|-----------|-------------|
| `--alignment_fasta <path>` | Chemin vers le fichier d'alignement au format FASTA (contenant les séquences cibles). |
| `--output_file <path>` | Chemin complet du fichier de sortie où seront écrites les signatures (extensions suggérées: .txt, .fasta). |

---

## 2. Géométrie & Structure Globale

| Paramètre | Défaut | Label Interface (FR) | Description |
|-----------|--------|----------------------|-------------|
| `--signature_max_length` | 400 | Longueur max signature | Longueur maximale totale de la signature LAMP (de F3 à B3). |
| `--min_signatures_for_success` | 1 | Nb Signatures Min | Nombre de signatures valides à trouver avant d'arrêter la recherche (si >1, le script continue). |
| `--min_primer_spacing` | 1 | N/A | Distance minimale (en bases) entre deux amorces adjacentes. |
| `--min_inner_pair_spacing` | 1 | N/A | Distance minimale entre F1c et B1c (Spacer central). |
| `--max_overlap_percent` | 0 | Chevauchement Max (%) | Pourcentage maximum de chevauchement autorisé entre deux signatures pour qu'elles soient considérées distinctes. |

---

## 3. Paramètres Thermodynamiques (Critiques)

| Paramètre | Défaut | Label Interface (FR) | Description |
|-----------|--------|----------------------|-------------|
| `--max_tm_diff` | 5.0 | Différence Tm Max (°C) | [NOUVEAU] Différence de Tm maximale autorisée entre deux amorces voisines (Filtre Thermique Dynamique). |
| `--dna_conc` | 400 | ADN (nM) | Concentration d'ADN (nM) pour le calcul du Tm (Primer3 / Owczarzy). |
| `--dntp_conc` | 1.4 | dNTP (mM) | Concentration de dNTP (mM). |
| `--salt_monovalent` | 50 | Sel Monovalent (mM) | Concentration de sels monovalents (mM) (ex: KCl). |
| `--salt_divalent` | 8 | Sel Divalent (mM) | Concentration de sels divalents (mM) (ex: Mg++). |
| `--thermodynamic_path` | /etc/primer3_config/ | N/A | Dossier contenant les tables thermodynamiques de Primer3. |
| `--primer3_executable` | /usr/bin/primer3_core | N/A | Chemin vers l'exécutable Primer3. |

## 7. Gestion de la Diversité & Mismatches (Avancé)

Ces paramètres permettent de gérer les alignements avec variabilité (virus, variants).

| Paramètre | Défaut | Label Interface (FR) | Description |
|-----------|--------|----------------------|-------------|
| `--entropy_threshold` | 1.5 | Seuil Entropie | [NOUVEAU] Seuil d'entropie (Shannon) au-dessus duquel une région est exclue. |
| `--min_base_frequency` | 0.05 | Fréquence min. bruit | [NOUVEAU] Fréquence minimale (5%) pour qu'une base soit incluse dans un code IUPAC. |
| `--max_total_mismatches` | 2 | Max Mismatches Total | Nombre maximum de mismatches autorisés par amorce contre une séquence cible. |
| `--max_consecutive_mismatches` | 2 | Max Mismatches Consécutifs | Nombre maximum de mismatches consécutifs (évite les bulles instables). |
| `--max_3prime_mismatches` | 2 | Max Mismatches 3' | Mismatches tolérés en 3' (zone critique pour la polymérase). |
| `--three_prime_zone_size` | 5 | Zone 3' (bases) | Taille de la zone 3' (en bases) surveillée plus strictement. |
| `--primer_min_match_percent` | 80.0 | % Match minimum | Pourcentage minimal de conservation requis pour une amorce stricte. |
| `--primer_iupac_min_percent` | 98.0 | % Match après IUPAC | Pourcentage de couverture requis si l'amorce contient des bases IUPAC (dégénérées). |
| `--min_primer_coverage` | 80.0 | % Élimination primer | Pourcentage des séquences de l'alignement qui doivent être reconnues par l'amorce. |

## 9. Paramètres Avancés (Hardcoded dans `lib/LLNL/LAVA/Core.pm`)

Ces paramètres ne sont pas modifiables en ligne de commande mais définissent le comportement interne du moteur LAVA 2026.

### Modèle de Pénalité (Sigmoïde)
### Modèle de Pénalité (Sigmoïde)
| Constante (CLI Option) | Valeur | Label Interface (FR) | Rôle |
|------------------------|--------|----------------------|------|
| `--penalty_plateau` | 0.25 | Plateau de Pénalité | Zone de "confort" autour de la distance cible où la pénalité est nulle (+/- 25%). |
| `--penalty_slope` | 0.15 | Pente Sigmoïde | Pente de la courbe de pénalité au-delà du plateau (plus c'est élevé, plus la pénalité monte vite). |
| `max_penalty` | 100 | N/A | Score de pénalité maximal possible pour un critère de distance. |

### ratios Géométriques (Target Distances)
Les distances cibles entre amorces sont calculées proportionnellement à la longueur totale de la signature.
- **F3-F2** : 12% de la longueur totale.
- **F2-F1** : 18% de la longueur totale.
- **Inner (F1c-B1c)** : 40% de la longueur totale.
